**ATHLETIX** is a sport-focused fitness tracker built as a static web application using **HTML, CSS, and JavaScript**. It helps users assess their fitness, choose a sport, and follow sport-specific workout plans designed for athletic performance.

The platform includes an exercise library with GIF demonstrations, live workout sessions with timers, and smart recovery pop-ups during rest breaks. An athletic fitness tracker monitors strength, speed, endurance, mobility, and consistency using browser LocalStorage.

ATHLETIX is designed to feel like a real fitness-tech startup product, focusing on clean UI, sporty design, and performance-driven user experience.
